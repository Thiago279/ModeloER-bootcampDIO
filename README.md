# E-commerce MER - Desafio de Projeto

## Descrição
Este repositório contém o Modelo Entidade-Relacionamento (MER) de um sistema de e-commerce, desenvolvido como parte de um desafio de projeto do bootcamp promovido pela [Digital Innovation One (DIO)](https://www.dio.me/) em parceria com a Heineken. O modelo foi criado utilizando o MySQL Workbench e visa representar um banco de dados para um sistema de vendas online.

## Objetivo do Desafio
O desafio consistiu em projetar um banco de dados que suporte as principais funcionalidades de um e-commerce, incluindo:
- Gerenciamento de clientes (Pessoa Física e Pessoa Jurídica).
- Registro de pagamentos por diferentes formas (cartão, boleto, PIX).
- Controle de estoque e produtos.
- Processamento de pedidos.

## Ferramentas Utilizadas
- **MySQL Workbench**: Utilizado para o design do MER.


## Aprendizados
Durante o desafio, foi possível aprender:
- A aplicação de especialização em modelagem de banco de dados.
- Como estruturar um banco de dados para atender às necessidades de um sistema real.
